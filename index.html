Теоретические вопросы

  1. Какие кавычки можно использовать для создания строк в JS и в чём разница между ними?

    Ответ:

    • Одинарные (' ')
    • Двойные (" ")
    • Обратные (` `)
    Разницы между одинарными и двойными кавычками при создании строк нет.
    А вот обратные дают нам новые возможности. Строка в таких кавычках превращается в шаблонную строку.
    В них можно вставлять произвольные выражения, обернув их в ${...}. Шаблонные строки могут содержать переносы строки.

  2. Какими методами можно найти подстроку в строке? Приведите пример.

    Ответ:

    • Метод indexOf() возвращает индекс (позицию) в первое вхождение указанного текста в строку.
      'Миша'.indexOf('М');
    • Метод lastIndexOf() возвращает индекс последнего вхождения указанного текста в строку
      'Миша'.lastIndexOf('и');
    • Метод search() — он тоже ищет строки по заданному значению и возвращает позицию совпадения.
      'Сейчас весна'.search('весна');
    • Метод includes выполняет проверку, есть ли указанный набор символов в строке.
      'Сейчас весна'.includes('весна');

  3. Самостоятельно разберитесь, зачем нужен специальный символ \n?

    Ответ: это так называемый «символ перевода строки».
    Позволяет сделать перенос текста в строке без использования обратных кавычек.

  4. Напишите код, который делает первый символ заглавным. Например, "настя" ⇒ "Настя"

    Ответ: const userName = 'валентина';
    console.log(userName[0].toUpperCase());

  5. Какой из вариантов округления делает это по математическим правилам?

    Ответ: Math.round() округляет до ближайшего целого числа.

  6. Как сгенерировать случайное число от 1 до 100?

    Ответ: console.log(Math.round(Math.random() * 100));

  7. Что выведет в консоль?
    const x="Солнце";
    console.log("The string length is " + x.length)

    Ответ: The string length is 6

  8. Каким будет результат в представленном ниже фрагменте кода?
    const upperText="ВелоСипед";
    document.write(upperText.toUpperCase())

    Ответ: должен выдать все заглавными буквами ВЕЛОСИПЕД. Но, как я поняла, только в момент загрузки страницы.

  9. Каким будет результат в приведенном ниже фрагменте кода?
    let str = "розы такие розы";
    str.replace("розы","красивые")

    Ответ: выведет "красивые такие розы"

  10. Что выведет в консоль?
    const str = "abcdefghij";
    console.log("(2,3): " + str.substr(2,3));

    Ответ: (2,3): cde
    Метод substr() возвращает указанное количество символов из строки, начиная с указанной позиции.
